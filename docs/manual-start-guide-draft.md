## ステップ1 / 5 — ログインする

**説明:**
登録時のメールアドレスとパスワードを入力して「ログイン」をタップしてください。

**使用画像:** 手順１.png（ログイン画面）

**切り抜き範囲:**
- 「店舗ログイン」見出し から「パスワードをお忘れですか？」までのカード全体
- 上部のYobuHoロゴは **残す**（ブランド視認のため）

**強調指示:**
- 「ログイン」ボタン（ワインローズ）にコーラル円 `#FF7A59` + 👆手カーソル
- パスワード入力欄には何も強調不要（目線を「ログイン」ボタンに集中）

## ステップ2 / 5 — ホテルを検索する

**説明:**
ご案内できるエリアを選ぶか、ホテル名・住所で直接検索できます。

**使用画像:** 手順２.png（検索画面全体）

**切り抜き範囲:**
- 上半分のみ切り出し: 「ホテル名で検索」入力欄 ＋ 「エリアから探す」日本全国ボタン（11地方）
- 「ご案内実績の登録数が多いほど…」のヒント枠は **残す**（動機付けになる）

**強調指示:**
- 検索窓に 👆手カーソル
- 「関東 9,002件」エリアボタン にコーラル円 + 矢印（任意の地方を選ぶことを示唆）

---

## ステップ3 / 5 — 「情報登録」をタップ

**説明:**
ご案内できるホテルが見つかったら、右下の「📝 情報登録」ボタンをタップします。

**使用画像:** 手順４.png（ホテル一覧）

**切り抜き範囲:**
- ホテルカード **1枚だけ** を切り抜き（例: ホテルサンルート東新宿）
- カード全体: ホテル名・住所・最寄駅・ビジネスタグ・情報登録ボタン

**強調指示:**
- 右下「📝 情報登録」ボタンにコーラル円 + 👆手カーソル

---

## ステップ4 / 5 — 呼べる / 呼べない を選ぶ

**説明:**
そのホテルにご案内できるかを選び、交通費があれば金額を入力してください。

**使用画像:** 手順３.png（ホテル情報登録モーダル）

**切り抜き範囲:**
- モーダル上半分: ホテル名ピンクバナー〜「交通費」入力欄まで
- 「メモ」「確認画面に進む」ボタンは **次ステップ用** に分ける

**強調指示:**
- 「✅ 呼べる」ボタンにコーラル円 + 👆手カーソル
- 交通費欄に薄い矢印（任意項目であることを示す補足）

**強調指示:**

## Canva作業時の共通スタイル指示（確定版・ブランド整合）

### カラーパレット（実画面に合わせた確定色）
- **背景色**: ウォームオフホワイト `#F8F5F0`
- **スクショカード背景**: 白 `#FFFFFF`
- **ブランドメイン**: ゴールド `#C9A961`（YobuHoロゴ・見出し下ライン・ページ番号）
- **サブアクセント**: ワインローズ `#A85A72`（ボタン枠・強調テキスト時）
- **強調色（矢印・円・手カーソル）**: コーラルオレンジ `#FF7A59`
  - 理由: ゴールド背景・白カードどちらに対してもコントラスト最強
- **本文テキスト**: ダークグレー `#222222`
- **補助テキスト（ステップN/5等）**: グレー `#888888`

### タイポグラフィ
- 見出し: Noto Sans JP **Bold** 28〜32pt
- 本文: Noto Sans JP **Regular** 14〜16pt（行間1.7）
- ステップ番号「ステップN / 5」: Inter Medium 12pt（ゴールド `#C9A961`）
- 英数字・金額: Inter

### レイアウト
- スクショカード: 丸角 20px + 薄い影 `0 4px 12px rgba(0,0,0,0.06)`
- カード上下左右に余白最低40px
- 1ページ=1ステップの鉄則。情報詰め込み禁止
- 手カーソル👆 + コーラル円で強調（Claudeインストールガイド風）

### ページサイズ
- A4縦（210×297mm） 印刷・PDF両対応
- 余白: 上下24mm、左右20mm
