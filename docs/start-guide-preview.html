SHOP START GUIDE
店舗管理 スタートガイド
最初のホテル情報を登録し、
YobuChatで訪問者と会話を始めるまで
約10分で完了

目次

  1. ログインするメール・パスワード
  2. ホテルを検索するエリア / キーワード
  3. 「情報登録」をタップホテル一覧から選ぶ
  4. 呼べる / 呼べない を選ぶ交通費も入力
  5. 確認して保存即反映
  6. YobuChatをONにする訪問者チャット開始
  7. 受付時間・定型文を設定自動応答もOK
STEP 01 / 07

ログインする

登録時のメールアドレスとパスワードを入力して「ログイン」ボタンをタップしてください。

🔐
ログイン画面のスクリーンショット
「ログイン」ボタンにコーラル円 + 👆手カーソルを配置
💡 ポイント：パスワードをお忘れの場合は「パスワードをお忘れですか？」から再発行できます。
STEP 02 / 07

ホテルを検索する

ご案内できるエリアを選ぶか、ホテル名・住所で直接検索できます。

🔍
検索画面 + エリアボタン
検索窓に👆手カーソル／関東等のエリアボタンにコーラル円
💡 ポイント：ご案内実績の登録数が多いほど、該当プラン内で上位に表示されます。
STEP 03 / 07

「情報登録」をタップ

ご案内できるホテルが見つかったら、右下の 📝 情報登録 ボタンをタップします。

🏨
ホテルカード（1枚だけ切り抜き）
右下「📝 情報登録」ボタンにコーラル円 + 👆手カーソル
STEP 04 / 07

呼べる / 呼べない を選ぶ

そのホテルにご案内できるかを選び、交通費があれば金額を入力してください。

✅
情報登録モーダル（上半分）
「✅ 呼べる」ボタンにコーラル円 + 👆手カーソル
💡 ポイント：交通費が発生しない場合は「交通費無料」にチェック。未入力のまま保存もOKです。
STEP 05 / 07

確認して保存する

「確認画面に進む」をタップすれば登録完了です。ポータルに即座に反映されます。

💾
情報登録モーダル（下半分）
「確認画面に進む」ボタンにコーラル円 + 👆手カーソル
🎉 お疲れさまでした！ ここまでで最初のホテル登録は完了です。続いてYobuChatを設定してみましょう。
PART 2
💬 YobuChatで訪問者と話す
STEP 06 / 07

YobuChatをONにする

管理画面の上部タブ「💬 YobuChat」をタップし、「チャット機能を有効にする」トグルをONにします。

💬
YobuChat管理画面（タブ＋有効化トグル）
「💬 YobuChat」タブにコーラル円／「有効にする」トグルに👆手カーソル
💡 ポイント：YobuChatをONにすると、ポータルのホテル詳細ページから訪問者がチャットで問い合わせできるようになります。
STEP 07 / 07

受付時間・定型文を設定

受付時間（例: 10:00〜翌3:00）と、よく使う返信（定型文）を登録しておくと対応が格段に楽になります。

⏰
受付時間設定 + 定型文追加フォーム
受付開始・終了時刻の入力欄／「定型文を追加」ボタンにコーラル円
💡 ポイント：通知先メールを設定すれば、新しい問い合わせが入った時に自動で通知が届きます。不在時も見逃しません。
